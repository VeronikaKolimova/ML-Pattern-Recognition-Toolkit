# ML-Pattern-Recognition-Toolkit
ML toolkit for anomaly detection (NYSE stocks), manifold learning (MDS/UMAP), and CNN/ResNet50 models achieving 89% accuracy in potato disease classification. / ML-инструментарий: обнаружение аномалий в акциях NYSE, снижение размерности (MDS/UMAP), CNN/ResNet50 для классификации болезней картофеля (89% точности).
